VGirol\JsonApiAssert\Asserts\Structure\AssertErrorsObject::assertIsValidErrorsObject

Asserts that a json fragment is a valid errors object.

It will do the following checks : 1) asserts that the errors object is an array of objects (@see assertIsArrayOfObjects). 2) asserts that each error object of the collection is valid (@see assertIsValidErrorObject).

VGirol\JsonApiAssert\Asserts\Structure\AssertStructure::assertIsValidIncludedCollection

Asserts that a collection of included resources is valid.

It will do the following checks : 1) asserts that it is an array of objects (@see assertIsArrayOfObjects). 2) asserts that each resource of the collection is valid (@see assertIsValidResourceObject). 3) asserts that each resource in the collection corresponds to an existing resource linkage present in either primary data, primary data relationships or another included resource. 4) asserts that each resource in the collection is unique (i.e. each couple id-type is unique).

VGirol\JsonApiAssert\Asserts\Structure\AssertMemberName::assertIsValidMemberName

Asserts that a member name is valid.

It will do the following checks : 1) asserts that the name is a string with at least one character. 2) asserts that the name has only allowed characters. 3) asserts that it starts and ends with a globally allowed character.
